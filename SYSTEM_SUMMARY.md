- **技术栈**: Vue 3 + Element Plus + ECharts
- **页面**:
  - `index.html` - 主界面（登录、仪表板、各功能模块）
  - `assessment.html` - 在线答题页面
  - `report.html` - 报告可视化页面
- **功能**:
  - 响应式设计
  - 完整导航
  - 实时数据更新

### 后端
- **技术栈**: Node.js + Express + SQLite
- **API路由**:
  - `server.js` - 主服务器、认证、项目管理
  - `phase2.js` - 第二期功能（数据上传、任务管理、报告）
  - `phase3.js` - 第三期功能（人才地图、继任计划、预测）
  - `assessment.js` - 答题功能（答题、提交、评分）
  - `tool-config.js` - 工具配置（工具管理、题目管理）
- **数据库**: SQLite，已初始化完整数据结构

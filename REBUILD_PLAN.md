# 人才测评系统重建方案（对标SHL/北森）

## 一、系统架构 redesign

- `competency_models` - 胜任力模型
- `competency_dimensions` - 胜任力维度

#### 报告模板
- `report_templates` - 报告模板

## 二、功能模块详细设计

### 2.1 胜任力模型管理
- 内置SHL UCF模型
- 内置北森53维度模型
- 支持自定义模型
- 岗位胜任力匹配

### 2.2 专业测评工具
1. **大五人格测评** (60题，对标SHL OPQ32)
   - 神经质、外向性、开放性、宜人性、尽责性
   - 每个维度6个子维度
   
2. **MBTI职业性格** (93题)
   - 外向/内向、感觉/直觉、思考/情感、判断/知觉
   
3. **DISC行为风格** (24题)
   - 支配型、影响型、稳健型、谨慎型
   
4. **霍兰德职业兴趣** (60题)
   - 现实型、研究型、艺术型、社会型、企业型、常规型
   
5. **EQ情商测评** (33题)
   - 情绪感知、情绪理解、情绪调节、情绪运用
   
6. **认知能力测评** (对标SHL Verify)
   - 数字推理、言语推理、逻辑推理
   
7. **工作价值观** (21题)
   - 成就感、独立性、利他主义等

### 2.3 360度评估
- 自定义评估问卷
- 多维度反馈收集
- 自评、上级、同事、下级评估
- 生成对比报告

### 2.4 人才盘点
- 绩效-潜力九宫格
- 团队人才分布
- 关键人才识别
- 继任风险分析

### 2.5 个人发展计划（IDP）
- 发展目标设定
- 发展行动规划
- 进度跟踪
- 效果评估

### 2.6 专业报告系统
- 个人测评报告
- 团队对比报告
- 岗位匹配报告
- 发展建议报告

## 三、实施计划

### Phase 1: 基础设施（1天）
- 重建数据库结构
- 实现胜任力模型
- 优化后端API

### Phase 2: 测评工具（2天）
- 实现7个专业测评工具
- 每工具60-93题
- 科学计分算法

### Phase 3: 高级功能（1天）
- 360评估
- 人才盘点
- IDP

### Phase 4: 报告系统（1天）
- 专业报告模板
- 可视化展示
- 导出功能

## 四、技术要点

### 4.1 测评计分算法
- 基于项目反应理论（IRT）
- 支持常模比对
- 计算百分等级

### 4.2 报告生成
- 基于模板的动态生成
- 支持图表可视化
- 多格式导出

### 4.3 数据安全
- 测评数据加密
- 权限控制
- 操作日志

## 五、对标SHL/北森的关键特性

1. **科学性** - 基于心理学理论
2. **专业性** - 专业报告格式
3. **全面性** - 覆盖人才管理全流程
4. **可定制** - 支持企业个性化
5. **可视化** - 直观的数据展示

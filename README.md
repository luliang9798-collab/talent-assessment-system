# 人才测评系统

对标 SHL/北森的专业人才测评系统

## 功能特性

- ✅ 28+ 专业测评工具（MBTI、大五人格、SHL胜任力等）
- ✅ 人才九宫格（绩效-潜力矩阵）
- ✅ 专业测评报告生成
- ✅ 用户管理与权限控制
- ✅ 测评数据可视化

## 一键部署

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/luliang9798-collab/talent-assessment-system)

### 部署步骤

1. 点击上方按钮
2. 连接 GitHub 账户
3. 点击 "Connect"
4. 等待部署完成（约 5 分钟）
5. 获得永久访问地址

## 本地运行

```bash
# 安装依赖
cd backend-nodejs
npm install

# 启动服务
node server-minimal.js

# 访问
http://localhost:3000
```

## 默认账号

- 管理员: admin / admin123
- 用户: user / user123

## 技术栈

- 后端: Node.js + Express + SQLite
- 前端: Vue 3 + Element Plus + ECharts
- 部署: Render (免费)

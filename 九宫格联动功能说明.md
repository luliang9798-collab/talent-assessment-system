# 测评数据与九宫格联动功能测试文档

## 功能概述

已实现测评数据与九宫格（9-Box Grid）的联动功能，用于人才盘点。九宫格是人才管理中的核心工具，通过两个维度评估员工：
- **X轴（绩效）**：员工过去的业绩表现（1-5分）
- **Y轴（潜力）**：员工未来的发展潜力（1-5分），基于测评数据自动计算

## 实现的功能

### 1. 数据库表结构

#### nine_box_grids（九宫格评估表）
- id: 主键
- grid_name: 九宫格名称
- grid_type: 评估类型（talent_review/succession_planning/high_potential）
- review_period: 评估周期
- description: 描述
- status: 状态（draft/active/archived）
- created_by: 创建人
- created_at/updated_at: 时间戳

#### nine_box_positions（九宫格位置表）
- id: 主键
- grid_id: 九宫格ID
- user_id: 员工ID
- performance_score: 绩效分数（1-5）
- potential_score: 潜力分数（1-5）
- grid_x: X轴位置（1-3）
- grid_y: Y轴位置（1-3）
- quadrant: 象限（1-9）
- notes: 备注
- created_at/updated_at: 时间戳

#### performance_records（绩效记录表）
- id: 主键
- user_id: 员工ID
- review_period: 评估周期
- performance_score: 绩效分数
- performance_level: 绩效等级
- kpi_score: KPI分数
- goals_achievement: 目标达成率
- competencies_score: 能力素质分数
- reviewer_id: 评估人
- review_date: 评估日期
- notes: 备注

### 2. 后端API接口

#### 九宫格管理
1. `POST /api/nine-box/grids` - 创建九宫格评估
2. `GET /api/nine-box/grids` - 获取九宫格列表
3. `GET /api/nine-box/grids/:id` - 获取九宫格详情
4. `POST /api/nine-box/grids/:id/positions` - 添加/更新员工位置
5. `GET /api/nine-box/grids/:id/positions` - 获取所有员工位置

#### 测评数据联动
6. `POST /api/nine-box/calculate-potential` - 根据测评数据计算潜力分数
7. `POST /api/nine-box/batch-calculate-potential` - 批量计算所有员工的潜力分数
8. `GET /api/nine-box/users/:userId/assessments` - 获取用户的测评结果

#### 绩效数据管理
9. `POST /api/performance` - 录入绩效分数
10. `GET /api/performance/:userId` - 获取员工绩效记录
11. `POST /api/nine-box/sync-performance` - 同步绩效数据到九宫格

#### 用户管理
12. `GET /api/users` - 获取用户列表
13. `GET /api/users/:id` - 获取用户详情

### 3. 前端页面

#### nine-box.html（九宫格展示页面）
- 创建九宫格评估
- 展示九宫格矩阵（3x3格子）
- 添加员工到九宫格
- 根据测评数据自动计算潜力分数
- 同步绩效数据
- 查看员工测评详情

#### index.html（主页面）
- 侧边栏添加"人才九宫格"菜单项
- 快速操作区添加"查看九宫格"按钮

## 测评数据与九宫格联动逻辑

### 潜力分数计算

潜力分数基于员工的测评结果自动计算，具体逻辑：

1. **数据来源**：
   - 大五人格测评（开放性、尽责性、外向性）
   - 情商测评（情绪管理、人际关系、自我认知）
   - 工作动机测评（成就动机、成长需求）
   - 工作价值观测评（组织匹配度）

2. **计算方法**：
   ```
   潜力分数 = (测评总分 / 100) × 5
   ```
   
   - 将测评分数（假设0-100分）转换为1-5的潜力分数
   - 如果员工有多个测评结果，取平均值
   - 如果员工没有测评数据，使用默认分数3

3. **API调用**：
   - 单个员工：`POST /api/nine-box/calculate-potential` with `{ user_id: 123 }`
   - 批量计算：`POST /api/nine-box/batch-calculate-potential` with `{ grid_id: 1 }`

### 绩效分数录入

绩效分数需要手动录入（因为系统无法访问公司的绩效系统），支持两种方式：

1. **手动录入**：
   - 调用 `POST /api/performance` 录入绩效分数
   - 支持KPI分数、目标达成率、能力素质分数等

2. **批量同步**：
   - 调用 `POST /api/nine-box/sync-performance` 从绩效记录同步到九宫格
   - 自动获取最新评估周期的绩效分数

### 九宫格矩阵

九宫格是一个3x3的矩阵：

```
        高潜力    中潜力    低潜力
高绩效   格子1    格子2    格子3
中绩效   格子4    格子5    格子6
低绩效   格子7    格子8    格子9
```

**人才管理策略**：
- **格子1（超级明星）**：重点保留，快速晋升，承担更大责任
- **格子2（核心人才）**：稳定发展，提供成长机会
- **格子5（中坚力量）**：组织的基础，需要关注和发展
- **格子9（问题员工）**：需要改进计划，或考虑淘汰

## 使用流程

### 1. 创建九宫格评估

1. 登录系统
2. 点击侧边栏"人才九宫格"
3. 点击"创建九宫格"按钮
4. 填写九宫格名称、评估类型、评估周期、描述
5. 点击"创建"

### 2. 添加员工到九宫格

1. 在九宫格详情页，点击"添加员工到九宫格"
2. 选择员工
3. 输入绩效分数（或留空稍后录入）
4. 输入潜力分数（或留空根据测评数据自动计算）
5. 点击"添加"

### 3. 批量计算潜力分数

1. 在九宫格详情页，点击"批量计算潜力分数"
2. 系统自动根据测评数据计算所有员工的潜力分数
3. 确认计算结果显示

### 4. 同步绩效数据

1. 先录入员工绩效分数（调用POST /api/performance）
2. 在九宫格详情页，点击"同步绩效数据"
3. 系统自动从绩效记录同步最新绩效分数到九宫格

### 5. 查看员工测评详情

1. 在九宫格详情页，点击员工标签
2. 在信息面板中，点击"查看测评详情"
3. 系统打开新窗口，展示该员工的所有测评报告和结果

## 注意事项

1. **测评数据要求**：员工需要完成至少一次测评，系统才能计算潜力分数
2. **绩效数据录入**：需要手动录入绩效分数，或从其他系统导入
3. **权限管理**：只有管理员和经理可以创建九宫格和评估员工
4. **数据更新**：当员工完成新的测评后，需要重新计算潜力分数
